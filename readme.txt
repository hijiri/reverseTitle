●記事タイトルをサイト名の前に表示するプラグイン

■概略
このソフトウェアは、Loggixを使用したサイトの<title>～</title>の値を変更して、記事タイトルをサイト名の前に表示する為のプラグインです。

■詳細
Loggixを使用したサイトでは、標準で「サイト名 : 記事タイトル」というタイトルになります。

これを「記事タイトル : サイト名」となる様にすると、検索エンジンで記事がヒットした時に、閲覧者の目に留まり易くSEO的に良い(らしい)です。

■インストール/アンインストール方法
インストール
    /plugins/へreverseTitle.phpをアップロードします。
アンインストール
    /plugins/からreverseTitle.phpを削除します。

■使用方法
インストールすれば何もしなくても直ぐに適用されます。
